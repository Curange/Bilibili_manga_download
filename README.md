# B站漫画下载器
哔哩哔哩漫画下载工具， 支持范围下载

***此版本还在开发中，欢迎给出您的建议和方案~***

## 项目特点
1：拥有图形界面（易用程度大幅度提升） <br />
2：支持多样的下载范围自定义 <br />
3：支持付费漫画下载（前提是已经购买的漫画）
